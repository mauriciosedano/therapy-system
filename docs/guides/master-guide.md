# Guía del Maestro

## 1. Comandos Disponibles

### 1.1 Pacientes
- "Registra nuevo paciente [nombre] [teléfono]"
- "Actualiza datos de [paciente]"
- "Consulta información de [paciente]"

### 1.2 Terapias
- "Registra terapia [tipo] para [paciente]"
- "Consulta terapias de [paciente]"
- "Muestra historial de [paciente]"

### 1.3 Fórmulas
- "Prepara fórmula [tipo] para [paciente]"
- "Consulta última fórmula de [paciente]"

## 2. Gestión de Aprendices

### 2.1 Permisos
- "Autoriza a [aprendiz] para consultas"
- "Permite que [aprendiz] registre terapias"
- "Revoca acceso a [aprendiz]"

### 2.2 Supervisión
- "Muestra actividad de [aprendiz]"
- "Revisa registros de [aprendiz]"
